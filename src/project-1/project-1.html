Проект Sedona

HTML + CSS + JS

Учебный проект сайта туристического города Седона. Проект выполнен по тренировочному макету HTML Academy.

В рамках этого проекта я использовал сборщик проектов Gulp и препроцессор Less. Сайт состоит из трех страниц: Главная, Фото и видео, Форма отзыва с всплывающими окнами при отправке формы. Все три страницы выполнены в трех разрешениях: для мобильного устройства, планшета и широкого монитора.

Проект оказался крайне полезным для изучения основ верстки сайтов. В ходе работы над были отработаны навыки HTML и CSS с использованием flexbox и grid. Интерактивная карта реализована с помощью google maps.
